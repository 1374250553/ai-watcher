# Agents

## 项目概述

AI 接入平台 —— 提供统一的 AI 能力接入与管理服务。

## 技术栈

- **前端**：Vue 3 + TypeScript + Vite
- **后端**：待定
- **数据库**：PostgreSQL + Prisma
- **包管理**：pnpm

## 开发规范

1. 每次修改代码后自动运行 lint
2. 使用 ESlint + Prettier 统一代码风格
3. 提交前确保类型检查通过
4. 遵循 Conventional Commits 提交规范

## 工作流

- 功能开发完成后先部署本地预览确认效果
- 需求相关自动调用 feature-design skill 生成文档
- 涉及项目文档生成时调用 project-wiki skill
